# CodeFlow 🚀

Онлайн платформа для обучения программированию с админ-панелью для управления заявками.

## 🔗 Live Demo

**Главный сайт:** https://codeflow-flow.web.app/  
**Админ панель:** https://codeflow-flow.web.app/admin/index.html

---

## 📚 Стек технологий

| Компонент | Технология |
|-----------|-----------|
| **Frontend** | HTML5, CSS3, JavaScript (Vanilla) |
| **Build Tool** | Vite |
| **Backend** | Firebase Firestore |
| **Авторизация** | Firebase Authentication |
| **Хостинг** | Firebase Hosting |
| **Шрифты** | Montserrat (Google Fonts) |

---

## 🎨 Дизайн

- **Цветовая схема:** Тёмная (#0B2545) с неоновым зелёным акцентом (#CAFFBF)
- **Адаптивность:** Полностью responsive (мобилка, планшет, десктоп)
- **Мобильное меню:** Бургер-меню для экранов < 480px

## ✨ Основные функции

### Для пользователей
✅ **Просмотр курсов** — каталог всех доступных курсов  
✅ **Информация о преподавателях** — биографии и опыт  
✅ **Прайс-лист** — стоимость и длительность курсов  
✅ **Отзывы** — реальные отзывы студентов  
✅ **FAQ** — часто задаваемые вопросы  
✅ **Форма заявки** — отправка контактных данных

### Для администраторов
✅ **Админ-панель** — защищённая авторизацией  
✅ **Список заявок** — все полученные заявки в таблице  
✅ **Дата и время** — время отправки каждой заявки  
✅ **Логаут** — безопасный выход

## 📡 Firebase Firestore коллекции

| Коллекция | Поля | Описание |
|-----------|------|---------|
| `course` | name, title, description, price, duration | Курсы |
| `teacher` | name, role, experience, imgUrl, title | Преподаватели |
| `price` | name, price, duration, description | Прайсинг |
| `review` | text, name, age, rating | Отзывы (1-5 звёзд) |
| `faq` | question, answer | FAQ |
| `requests` | name, email, phone, message, timestamp | Заявки от пользователей |

## 🎯 Как начать

### Просмотр сайта
1. Перейди на https://codeflow-flow.web.app/
2. Просмотри курсы, преподавателей, цены
3. Заполни форму "Готов начать?" с контактными данными

### Вход в админ-панель
1. Перейди на https://codeflow-flow.web.app/admin/
2. Введи email: `admin@tami.com`
3. Введи пароль: `Admin123!`
4. Просмотри все поступившие заявки в таблице
5. Нажми "Выход" для выхода

## 🛠️ Локальная разработка

```bash
# Установка зависимостей
pnpm install

# Запуск dev сервера
pnpm run dev

# Сборка для продакшена
pnpm run build

# Деплой на Firebase
firebase deploy
```
