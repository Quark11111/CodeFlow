# CodeFlow 🚀

Онлайн платформа для обучения программированию с админ-панелью для управления заявками.

## 🔗 Live Demo

**Главный сайт:** https://codeflow-flow.web.app/  
**Вход в админ:** https://codeflow-flow.web.app/admin/  
**Админ-панель:** https://codeflow-flow.web.app/dashboard/

---

## 📚 Стек технологий

| Компонент | Технология |
|-----------|-----------|
| **Frontend** | HTML5, CSS3, JavaScript (Vanilla) |
| **Build Tool** | Vite |
| **Backend** | Firebase Firestore |
| **Авторизация** | Firebase Authentication |
| **Хостинг** | Firebase Hosting |
| **Шрифты** | Montserrat (Google Fonts) |

---

## 🎨 Дизайн

- **Цветовая схема:** Тёмная (#0B2545) с неоновым зелёным акцентом (#CAFFBF)
- **Адаптивность:** Полностью responsive (мобилка, планшет, десктоп)
- **Мобильное меню:** Бургер-меню для экранов < 480px

## ✨ Основные функции

### Для пользователей
✅ **Просмотр курсов** — каталог всех доступных курсов  
✅ **Информация о преподавателях** — биографии и опыт  
✅ **Прайс-лист** — стоимость и длительность курсов  
✅ **Отзывы** — реальные отзывы студентов  
✅ **FAQ** — часто задаваемые вопросы  
✅ **Форма заявки** — отправка контактных данных  
✅ **Адаптивный дизайн** — работает на всех устройствах

### Для администраторов
✅ **Авторизованная админ-панель** — защита от несанкционированного доступа  
✅ **Список всех заявок** — таблица с поступившими запросами  
✅ **Полная информация о клиенте** — имя, email, телефон, сообщение  
✅ **Дата и время отправки** — когда была отправлена заявка  
✅ **Безопасный логаут** — выход из системы

## 📡 Firebase Firestore коллекции

| Коллекция | Поля | Описание |
|-----------|------|---------|
| `course` | name, title, description, price, duration | Курсы программирования |
| `teacher` | name, role, experience, imgUrl, title | Преподаватели |
| `price` | name, price, duration, description | Прайсинг курсов |
| `review` | text, name, age, rating | Отзывы студентов (1-5 звёзд) |
| `faq` | question, answer | FAQ вопросы и ответы |
| `requests` | name, email, phone, message, timestamp | Заявки от пользователей |

## 🛠️ Локальная разработка

### Установка

```bash
# Установка зависимостей
pnpm install

# Запуск dev сервера
pnpm run dev
# Открыть http://localhost:5173
```

### Сборка и деплой

```bash
# Сборка для продакшена
pnpm run build

# Просмотр собранного проекта
pnpm run preview

# Деплой на Firebase
firebase deploy
```

### Переменные окружения (.env)

```
VITE_FIREBASE_API_KEY=xxxxx
VITE_FIREBASE_AUTH_DOMAIN=xxxxx
VITE_FIREBASE_PROJECT_ID=codeflow-flow
VITE_FIREBASE_STORAGE_BUCKET=xxxxx
VITE_FIREBASE_MESSAGING_SENDER_ID=xxxxx
VITE_FIREBASE_APP_ID=xxxxx
```

## 📧 Обработка заявок

Каждая заявка сохраняется в Firestore с информацией:
- 👤 Имя и фамилия
- 📧 Email адрес
- 📱 Номер телефона
- 💬 Текст сообщения
- ⏰ Дата и время отправки

Администратор видит все заявки в защищённой панели `/dashboard/`
